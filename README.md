Script voltado para avaliação de impacto utilizando modelos de crescimento. 

Os modelos individuais são:

- Gompertz
- Logístico
- Richards

Utilizando o método stacking, foi gerado o script para avaliação de impacto de determinado tratamento sobre uma população. Como teste, foram utilizados dados de covid. Alguns ajustes estão sendo feitos para melhora dos resultados e estimações.
